## Programming Notes

リストの「右」から要素を取る。スライスのときは添字がリストからはみ出ても
`IndexError` が生じない。

```python
>>> L = list(range(34))
>>> L[-1:-5:-1]
[33, 32, 31, 30]
>>> L[-31:-35:-1]
[3, 2, 1, 0]
>>> L[-33:-37:-1]
[1, 0]
>>> L[-35:-39:-1]
[]
```
