[Shell Features - Linux Command Line Computing
](https://learnbyexample.github.io/cli-computing/shell-features.html) 続きを読む。

* Redirection はコマンドのどこにでも書ける（先頭でもよい）
* `set noclobber` で `>` による既存ファイルへの上書きを阻止できる
  * それでも `>|` で上書きすることが可能
* 標準出力と標準エラー出力に同時に redirect する記号 (`&>`, `&>>`, `|&`) がある
* Here strings は最近まで知らなんだ

昔と違って、おそらく `>&` は覚えなくていい。苦手な理由はこれだった。
