13:20 Docker をやろう。Docker Hub にプッシュできるタグの付け方を習得するのだ。

```console
docker tag pushdemo:latest showayojyo/pushdemo:latest
docker push showayojyo/pushdemo:latest
```

次に `Dockerfile` の `FROM` を `scratch` に置き換えて新規イメージを構築する。最
初にやりたかった方式だ。

```Dockerfile
FROM scratch
WORKDIR /images
COPY *.png /images/
CMD ["/no/such/command"]
```

`docker` コマンドの組み方に慣れてくる：

```console
docker build -t "showayojyo/scratch-demo:0.0"
docker run --name demo "showayojyo/scratch-demo:0.0"
docker cp "demo:/images/sample.png" ./sample-cp.png
```

成果物を Docker Hub に送る。ブラウザー UI でのリポジトリー作成処理は不要だった：

```console
docker push showayojyo/scratch-demo:0.0
```

これがこうなる：
