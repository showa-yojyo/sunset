プとして配置したい。ただちに確認したい事項を次に記す：

* 手許の環境で `docker` コマンドは機能するか
* 適当なイメージを稼働できるか
* 不要なイメージを削除できるか
* 簡素なイメージを作成できるか
* 自作イメージを Docker Hub に配備できるか

まず、`docker --version` などのイメージと関係のないコマンドが成功することを確か
める。Docker 公式文書にあるように、`hello-world` イメージをダウンロード、稼働す
る試みも成功する。

```console
docker run hello-world
```

用が済んだのでこのイメージを削除したい。コンテナーを削除しないとイメージを基本的
には削除できない。`docker ps -a` でコンテナー ID を確認してから次のようにする：

```console
docker rm xxxxxxx
docker rmi hello-world
```
