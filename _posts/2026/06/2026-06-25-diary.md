Docker いじり。コンテナーを走らせずにイメージからファイルを抽出できることを知
る。<https://stackoverflow.com/questions/44769315/how-to-see-docker-image-contents>

```console
docker create "$TARGET_IMAGE" dummy-command
docker export -o image.tar "$(docker ps -lq)"
```

イメージを生成せずに、成果物に含まれることになるファイルを確認することができる：

```console
docker build -t "$TARGET_IMAGE" --output type=tar,dest=image.tar .
tar -tf image.tar
```
