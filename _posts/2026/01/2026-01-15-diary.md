時間があるので WSL Ubuntu に Python 3.14 環境を用意する。Miniconda 環境を作り直
すのが望ましいが、ここにある Python 3.13 環境にまだ用がある。

```console
mamba update mamba
mamba update --all

mamba create -c conda-forge -n python-3.14 python=3.14
eval "$(mamba shell hook --shell bash)"
mamba activate python-3.14

#mamba install ruff docutils jinja2 scrapy python-dateutil pyopenssl pyyaml bs4
#pip install jpholiday
```

Bash 開始時にシェルに読み込ませるファイルの Miniconda 初期化部分をコマンド

```console
mamba shell init --dry-run
```

が出力するコードに置き換える。ただし、コードを部分的に改造する。

```bash
# >>> mamba initialize >>>
# !! Contents within this block are managed by 'mamba shell init' !!
export MAMBA_EXE="$XDG_DATA_HOME/conda/envs/python-3.14/bin/mamba"
export MAMBA_ROOT_PREFIX="$XDG_DATA_HOME/conda"
__mamba_setup="$("$MAMBA_EXE" shell hook --shell bash --root-prefix "$MAMBA_ROOT_PREFIX" 2> /dev/null)"
if [ $? -eq 0 ]; then
    eval "$__mamba_setup"
else
    alias mamba="$MAMBA_EXE"  # Fallback on help from mamba activate
fi
unset __mamba_setup
# <<< mamba initialize <<<
mamba activate python-3.14
```

もう `conda` は使わないのならば Miniconda ディレクトリーを削除する。

```console
rm -rf ~/.local/share/miniconda3
```

<details><summary>ドラクエ 3 HD-2D 二周目プレイ記録：バラモスと一騎打ち</summary>
<p>水面のキラキラ全回収を続ける。ジパング辺りから。順序を記しにくいのが海の嫌なところの一つだ。</p>

<p>すべてを回収したと信じてバラモス戦に戻る。勇者レベル 48 一人で戦ってみる。
数戦試行錯誤した結果、ギガデイン主体で行くのが良かろうと判断。テドン北西の岩礁で拾ったさざなみの杖をフル活用する。
自動回復潰し目的でやいばのよろいを着込む。武器はグリンラッド東の高台で拾ったふぶきのつるぎ。
本番直前に溜めに溜めた急所のタネ類を注ぎ込む。特に、バラモスに後れを取らぬようすばやさを上げる。220 くらいでいい気がする。
空いたアクセサリー枠は氷のリング二丁。</p>
