ぎ込む。手作業だとキツイのでAPI を用いる：

```bash
if [[ $# -lt 1 ]] ; then
    echo Usage: ${0##*/} ACCOUNT/REPO >&2
    exit 2
fi

declare -r target="$1"

gh api --method PUT \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  /user/starred/${target}
```
